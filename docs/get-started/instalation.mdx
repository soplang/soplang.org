# Installation

This guide shows two easy ways to get started with Soplang:  
**downloading a prebuilt release** or **using the official Docker image**.

<HorizontalLine />

## General Download (All Platforms)

1. **Download:**  
   Grab the latest release for your platform from the [downloads page](#).

2. **Extract / Install:**  
   Unzip or install the package as provided by the release.  
   If it‚Äôs a single binary, place it somewhere in your `PATH` (e.g., `~/bin` on UNIX-like systems).

3. **(Optional) Add to PATH:**  
   Ensure the `soplang` binary is available on your PATH. For example (bash/zsh):

   ```bash
   echo 'export PATH="$HOME/bin:$PATH"' >> ~/.bashrc && source ~/.bashrc
   ```

4. **Verify Installation:**  
   Open your terminal and run:

   ```bash
   soplang --version
   ```

<HorizontalLine />

## Use Docker (No Local Install)

If you already have Docker, you can run Soplang without installing anything else.

1. **Pull the Image:**

   ```bash
   docker pull soplang/soplang:latest
   ```

2. **Run an Interactive Session:**

   ```bash
   docker run -it --rm soplang/soplang
   ```

3. **Run a Local File:**  
   Mount your project directory and execute a file:

   ```bash title="From your project directory"
   docker run -it --rm -v "$PWD":/app -w /app soplang/soplang soplang hello.sop
   ```
