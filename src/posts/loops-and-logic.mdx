Control flow is the heart of any logic.

## If Statements (Hadii)

```sop
hadii (da >= 18) {
    qor("Waad codeyn kartaa");
} hadii kale {
    qor("Ma codeyn kartid wali");
}
```

## Loops (Ku Celi)

```sop
ku_celi (x = 0; x < 10; x++) {
    qor(x);
}
```
