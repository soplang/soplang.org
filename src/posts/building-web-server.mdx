Soplang isn't just for scripts; it's for the web too!

## Setting up a Server

Using the `net` module (coming soon in v2.0 standard lib):

```sop
keen "net" sida net;

server = net.sameeServer((codsi, jawaab) => {
    jawaab.dir("Salaan sare!");
});

server.dhegayso(3000);
```
