# Arrays

Arrays waxa lagu qeexaa erayga `teed`. Waxay hayaan uruurin qiimayaal ah oo leh habab badan: maarayn, beddelid, iyo falanqayn.

## Declaring Arrays

```soplang
teed numbers = [1, 2, 3, 4]
teed magacyo = ["Ali", "Zahra"]
```

## Core Array Methods

### `dherer()` – Get Length

```soplang
teed x = [10, 20, 30]
qor("Length: " + x.dherer())  // 3
```

### `kudar(x)` – Add to End

```soplang
teed x = [1, 2]
x.kudar(3)   // [1, 2, 3]
x.kudar(4)   // [1, 2, 3, 4]
```

### `kasaar()` – Remove Last

```soplang
door last = x.kasaar()
qor("Removed: " + last)
```

### `kudar(teed)` – Concatenate

```soplang
teed a = [1, 2]
teed b = [3, 4]
teed c = a.kudar(b)    // [1, 2, 3, 4]
```

### `leeyahay(x)` – Contains Check

```soplang
haddii (c.leeyahay(3)) {
    qor("Waa jiraa")
}
```

## Utility Methods

### `nuqul()` – Copy Array

```soplang
teed asal = [10, 20]
teed copy = asal.nuqul()
qor("Copy: " + copy)
```

### `rog()` – Reverse

```soplang
asal.rog()
qor("Reversed: " + asal)
```

### `nadiifi()` – Clear

```soplang
copy.nadiifi()
qor("Emptied copy: " + copy)
```

### `habee()` – Sort

```soplang
teed t = [4, 2, 3]
t.habee()
qor("Sorted: " + t)
```

## Transform and Filter

### `aaddin(funk)` – Map Items

```soplang
hawl laban(x) {
    celi x * 2
}

teed nums = [1, 2, 3]
teed doubled = nums.aaddin("laban")
```

### `shaandhee(funk)` – Filter Items

```soplang
hawl is_even(x) {
    celi x % 2 == 0
}

teed filtered = nums.shaandhee("is_even")
```

## Indexing and Slicing

### `muuji(x)` – Find Index

```soplang
abn i = nums.muuji(2)
qor("Index of 2: " + i)
```

### `jar(a, b)` – Slice Sublist

```soplang
teed subset = nums.jar(1, 3)
qor("Subset: " + subset)
```

**Gunaanad:** `teed` waa qaab-dhismeed aasaasi ah oo Soo-maali-first ah, fududeynaaya maaraynta iyo ku celcelinta liisaska.
